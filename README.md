**Engine** is an automation-catered utility content datapack that uses lava as the main power source

This datapack introduces several new blocks, new choices for automation, utility tools, crafting items, and more!

`For Java Minecraft 1.21.10+`

## Features
Feature requests are always open and very welcome!
- Machines that actively consume lava to work
- Custom crafting and blueprints
- Standard modding items like wrenches and hammers
- **No Resourcepack required!**

## Getting Started
Craft a Lava-leaking Guidebook a single book and lava bucket. The bucket will be consumed.

The guide is in no way required for progression, but if you're only getting started it's recommended for some beginner guidance.
